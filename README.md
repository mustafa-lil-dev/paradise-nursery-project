# Paradise Nursery Shopping Application

This is a React-based e-commerce application for an online plant shop called Paradise Nursery. 
It allows users to browse a collection of 18 unique houseplants across three categories, 
add them to a shopping cart, and manage their selections using Redux for state management.

### Features:
- Landing page with a "Get Started" button.
- Responsive product listing with "Add to Cart" functionality.
- Cart management (increase/decrease quantity, delete items).
- Dynamic cart icon and total cost calculation.
